# huawei-vrp

Expert on Huawei CloudEngine S57xx and S67xx series switches running VRP (Versatile Routing Platform).

## Install
```bash
npx skills add https://github.com/lvacek2026/skills --skill huawei-vrp -g
```

## Contents

- VRP syntax rules (system-view, display, save)
- Port types (GigabitEthernet, XGigabitEthernet, 40GE/100GE)
- VLAN configuration (Access, Trunk, Hybrid ports)
- Eth-Trunk / LACP setup
- iStack stacking configuration
- STP / MSTP setup
- SSH access (RSA keys, AAA, VTY)
- `display` output diagnostics (ports, STP, Eth-Trunk)
- Useful diagnostic commands reference
- Quick-start template for new switch (`references/quickstart-template.md`)
